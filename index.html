What is Bouldering?

Bouldering is an increasingly poupular rock climbing sport with no ropes or harnesses. Indoor bouldering involves climbing walls with different types of artificial hand-holds attached to them. The routes are then designed by an experienced climber who selects a starting point, end point and a number of holds that they are allowed to touch in between.

Climbing requires a strong core, immense finger strength and concentration to hang on to the wall. Beside the physique, the mind is also at work to get to the finish point.

OUR GOALS


Practicability & Sociability

Our goal in this project is to simplify and modernize the climbing sport in UNNC. We want to create a system where communication between climbers and detailed feedback on the climbing routes are top priorities. We plan to include several convenient and effective features to make the software user friendly and resourceful.

Android Based

Fully responsive Android Application providing many features

Communicative

Multiple feedback features, such as free flow commentary section, reply rating, and others

User Friendly

Modern implementation of the application with easy to use UI interface

Language Supportive

Ability to support multiple languages, such as English and Chinese

Secure

Quality assurance over all feauters, including feedback, data collection and portability

Up To Date

Latest information about the routes and maintenance over feedback section
