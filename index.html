OUR GOALS


Practicability & Sociability

A software that allows climbers and climbing sport managers to browse, utilize, create, share, and collect data, leave comments on bouldering routes for each bouldering wall. Furthermore, this system aims to act as an interface to each bouldering wall, which means it can accept and deal with various types of walls and make them available to users. Besides, simple and practical UI design will provide user different options to select their preferences. Users even can make friends and communicate by this software …… More interesting functions will be implemented later :)
